✅ **Added/Updated:**
- Blue markers for all stations

- Choice between Apple Maps and Google Maps
- Compass moved to top right
- Cleaner, simpler interface

## 🚀 Quick Start

1. **Make the launcher executable** (one time only):
```bash
chmod +x /Users/mitchelldown/Documents/QLDFuelWatch/start-local-server.command
```

2. **Double-click** `start-local-server.command` in Finder

That's it! Your browser will open automatically.

## 📱 Features

- **Live QLD fuel prices**
- **Cheapest station highlighted in green**
- **Navigate with Apple or Google Maps**
- **Search by suburb**
- **Weather display**
- **Clean, simple interface**

## 🗺️ Navigation

When you click the navigate button (📍):
- **OK** = Apple Maps
- **Cancel** = Google Maps
